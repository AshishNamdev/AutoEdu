**Supported Portal**
--
- [UDISE](https://sdms.udiseplus.gov.in/)
- [MPBSE](https://mpbse.mponline.gov.in/MPBSE/MPBSE)
- [Education Portal 3.0](https://sederp.educationportal3.in/)
